🏥 Hospital Care AI Chatbot & Appointment System

An intelligent Healthcare Chatbot & Appointment Management System built using Python (Flask) that enables patients to book appointments, check doctor availability, and receive symptom-based guidance through a conversational interface.

---

📌 Project Overview

This project is designed to simplify hospital operations and enhance patient experience by automating appointment booking and providing AI-based assistance.

Users can:

- Book doctor appointments
- Check doctor availability
- Get symptom-based suggestions
- Manage appointments easily

---

🚀 Features

- 🤖 AI Chatbot for healthcare assistance
- 📅 Appointment booking system
- 👨‍⚕️ Doctor availability checking
- 🔐 User authentication (Login/Register)
- 🗄️ SQLite database integration
- 📊 Basic admin & patient workflow

---

🛠️ Tech Stack

- Backend: Python, Flask
- Frontend: HTML, JavaScript
- Database: SQLite
- Security: Password hashing

⚙️ Installation & Setup

1. Clone Repository

git clone https://github.com/your-username/hospital-chatbot.git
cd hospital-chatbot

2. Install Requirements

pip install flask

3. Run Project

python app.py

4. Open Browser

http://127.0.0.1:5000

---

💡 How It Works

1. User logs in or registers
2. Chatbot processes user queries
3. Detects intent (booking / availability / symptoms)
4. Stores appointments in database
5. Returns response instantly
